# Estrutura de Gestão: O Ciclo PDCA integrado ao 5W2H.

Este guia estabelece um fluxo de trabalho padronizado para a execução de projetos, garantindo que o planejamento estratégico (PDCA) seja operacionalizado com precisão técnica (5W2H).

O Ciclo PDCA (Planejamento Estratégico)O ciclo PDCA é a base para a melhoria contínua de processos:

1. P - Plan (Planejar): Estabeleça metas, métodos e planos de ação claros. Esta etapa envolve identificar problemas, analisar dados e definir indicadores de desempenho.
2. D - Do (Fazer/Executar): Implemente o plano de ação, preferencialmente em pequena escala, treinando a equipe e coletando dados sobre o processo.
3. C - Check (Verificar/Checar): Compare os resultados obtidos com as metas planejadas. É a fase de monitorar, avaliar os dados coletados e identificar desvios.
4. A - Act (Agir/Ajustar): Tome medidas corretivas baseadas nos resultados da verificação. 
 
Se o plano funcionou, padronize-o; caso contrário, reinicie o ciclo com novos planos.

Adição da Estrutura 5W2H (Operacionalização).

Para que a etapa "P - Plan" do PDCA seja eficaz, ela deve ser detalhada através da matriz 5W2H, que responde às perguntas cruciais para a execução:

# Elemento + Definição 

1. What (O quê): A ação ou meta a ser alcançada.
2. Why (Por que): A justificativa da ação.
3. Where (Onde): Local de execução.
4. When (Quando): Prazo ou cronograma.
5. Who (Quem): Responsável pela execução.
6. How (Como): Método ou passos a seguir.
7. How much (Quanto custa): Custo envolvido.
